## Mastery Template Docs Draft

```
==Description==
This template displays the mastery icon for the requested skill, plus the name as a link to the mastery page.

==Use==
<pre>{{Mastery|name of skill|width (optional)}}
```

### Examples

```
{{Mastery|Combat}}
{{Mastery|Farming|width=48}}
```
